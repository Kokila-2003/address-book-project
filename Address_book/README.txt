Project Name: Address Book Application

Student Name: Kokila A

Description:
This project implements an address book using C programming.
It supports adding, searching, editing, deleting contacts and
stores data in a CSV file using file handling.

Files Description:
main.c        - Program entry point
contact.c    - Contact related operations
contact.h    - Contact structure and function declarations
file.c       - File handling functions
file.h       - File function declarations
Contacts.csv - Stores contact details

Compilation:
gcc *.c

Execution:
./a.out
